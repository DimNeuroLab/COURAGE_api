English algorithms predictions overview

On this page we provide an overview over predictions generated with algorithms that were used inside the API demo (mainly twitter demo). Some examples demonstrate that the algorithms are not working perfectly and therefore can help to understand limitations of such tools. The sample data are taken from the respective test sets the algorithms were trained on. The current overview page includes examples for English sentiment, English emotion and English hate speech models.
